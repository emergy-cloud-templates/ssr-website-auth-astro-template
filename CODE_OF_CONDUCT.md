# Code Of Conduct

This project should be a practical and respectful place to collaborate.

## Expected Behavior

- Be direct, kind, and specific.
- Assume good intent while still asking for clarity.
- Keep technical criticism focused on code, docs, behavior, and tradeoffs.
- Make space for new contributors to ask setup questions.
- Respect maintainers' time and scope decisions.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information.
- Sharing secrets, credentials, logs, or Terraform state from systems you do not
  own.
- Repeated off-topic or bad-faith comments.

## Enforcement

Maintainers may edit, hide, or delete comments; close issues; block users; or
report abuse to GitHub when needed. The goal is to keep the project useful and
safe for contributors.

## Reporting

Report conduct concerns privately to the maintainer through the contact details
on their GitHub profile.
